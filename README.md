# AppNotas

Aplicación móvil para cálculo de promedio de notas desarrollada en Android Studio.

## Funcionalidades
- Login de usuario
- Registro de usuario
- Ingreso de 6 notas
- Cálculo automático del promedio
- Resultado aprobado o desaprobado

## Tecnologías utilizadas
- Android Studio
- Java
- XML
- SQLite
